【実装した基本機能】：

1. プレイヤーのギルド登録
   - 入力されたプレイヤー情報（名前、職業、HP、ATK、DEF）をギルドメンバーに追加。
   - 職業選択機能（戦士、魔法使い、弓使い、盗賊、その他）

2. プレイヤーのギルド退会
   - プレイヤー名を指定してギルドから削除。

3. ギルドメンバー一覧の簡易表示
   - 登録された全メンバーの名前と職業を一覧で表示。

4. ギルドメンバー1人分の詳細表示
   - 指定されたプレイヤーの詳細情報（名前、職業、HP、ATK、DEF）を表示。

----------------------------------
【実装した追加機能】（加点対象）：

5. 入力ミスやエラーへの対処
   - 存在しないプレイヤー名の検索や退会時に「該当者なし」と表示。
   - プレイヤー重複登録時に警告メッセージを表示。
   - 入力値の検証（HP > 0, ATK >= 0, DEF >= 0）
   - 無効な選択肢に対するエラーメッセージ

6. Menuクラスの作成
   - メニュー表示とユーザー操作処理をMenuクラスに分離し、main関数の可読性を向上。
   - 各機能を独立したメソッドとして実装。

7. ギルドメンバー情報のソート機能（追加機能）
   - HP（レベル）順に並び替えたリストを表示。
   - 降順ソート（高いレベルから低いレベル）

8. ギルドメンバーの検索機能
   - 名前の一部を入力することで曖昧検索が可能。
   - 大文字小文字を区別しない検索。

9. ギルド統計機能（新規追加）
   - 現在のギルドメンバー数を表示。
   - ギルドの状態を表示（活発/空）

10. ユーザーインターフェースの改善
    - 美しいメニューデザイン
    - 操作後の確認メッセージ
    - 画面クリア機能
    - 統一された出力フォーマット

----------------------------------
【使用方法】：

1. 実行ファイルを起動すると、メニューが表示されます。
2. 数字を入力して各機能を選択します。
3. 指示に従ってプレイヤーを登録・削除・表示等を行ってください。

【メニュー項目】：
1. メンバー登録 - 新しいプレイヤーをギルドに登録
2. メンバー退会 - 指定したプレイヤーをギルドから削除
3. メンバー一覧 - 全メンバーの簡易表示
4. メンバー詳細 - 指定したプレイヤーの詳細情報
5. レベル順一覧 - HP順にソートされたメンバー一覧
6. メンバー検索 - 名前の一部でメンバーを検索
7. ギルド統計 - ギルドの統計情報を表示
8. 終了 - プログラムを終了
